# What is a Content Delivery Network? 

A Content Delivery Network (CDN) is a system of distributed servers that work together to deliver web content to the end users, without long loading times.

## CDN caches content

CDNs rely on caching, which means that it temporarily stores copies of files in the servers around the world. This allows you to access content from a server near you, without long loading times.
