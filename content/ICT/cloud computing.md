# Cloud Computing

Cloud computing is all about delivering computing services like servers, storage, databases, networking, software, and more over the internet. Instead of owning and managing physical hardware, you rent what you need from cloud providers and pay based on usage.

## Advantages
* Flexible
* Scalable
* No need for own hardware
