**What is Azure Information Protection (AIP)?**

Azure Information Protection (AIP) is a Microsoft solution designed to help organizations classify, label, and protect sensitive data such as emails and documents. While the standalone AIP client and Office add-in were officially retired in [April 2024](https://techcommunity.microsoft.com/blog/microsoft-security-blog/retirement-notification-for-the-azure-information-protection-unified-labeling-ad/3791908), the core features still exist today as part of the [Microsoft Purview Information Protection platform](https://learn.microsoft.com/en-us/azure/information-protection/what-is-information-protection).

Instead of using the legacy AIP tools, organizations now rely on built-in sensitivity labels in Microsoft 365 apps like Word, Excel, and Outlook. These labels apply encryption, permissions, and visual markings—automatically or manually—based on the content or user actions.
