# What is Software as a Service (SaaS)

A way of delivering applications remotely over the internet, where the software is hosted by a third-party provider instead of on the customer’s premises. SaaS is one of [[three main cloud service models]].

## Examples
* Google Workspace
* Dropbox
* Slack
* Discord

## Advantages
* Scalability
* Cost savings (no need to host own servers)
* Access from anywhere

## Disadvantages

* Data security and compliance considerations
* [[Vendor lock-in]]
* Needs stronger access control
