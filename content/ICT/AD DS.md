# What is AD DS?
AD DS (Active Directory Domain Services) is a Microsoft service that provides centralized management of user, computers, and other resources within a network. It allows users to sign in with a single set of credentials, enables administrators to control access permissions, and simplifies the deployment of policies across groups. AD DS is a core component of Windows Server environments and domain-based network structures.
